# JavaScript基础
## JavaScript介绍
JavaScript ： 简称JS   
JS 分为三个部分：   
1. ECMAScript 标准 —— js的基本的语法
2. DOM —— Document Object Model 文档对象模型
3. BOM —— Browser Object Model 浏览器对象模型
### JavaScript是什么？
> JavaScript是以中国运行在 ***客户端*** 的 ***脚本语言***

是一门脚本语言  
是一门解释性语言
是一门动态类型的语言  
是一门基于对象的语言  
### JS的作用？
> 解决用户和浏览器之间交互的问题
### JS可以做什么？
#### **JavaScipt应用场景**
发展到现在几乎无所不能   
1. 网页特效
2. 服务端开发(Node.js)
3. 命令行工具(Node.js)
4. 桌面程序(Electron)
5. App(Cordova)
6. 控制硬件-物联网(Ruff)
7. 游戏开发(cocos2d-js)
#### **JavaScript与HTML和CSS的区别**  
HTML： 提供网页的结构，提供网页中的内容
CSS： 用来美化网页
JavaScript： 可以用来控制网页内容，给网页增加动态的效果
